# nginx-proxy demonstration

## これは何？
dockerコンテナのnginx-proxyの実行例をまとめたものです。

## ディレクトリ構成
※ディレクトリのみを表示しています。
```
.
└── VirtualHost  # バーチャルホストの実行例があるディレクトリ
    ├── public-html-01 # ウェブサイト１のドキュメントルート
    └── public-html-02 # ウェブサイト２のドキュメントルート
```
